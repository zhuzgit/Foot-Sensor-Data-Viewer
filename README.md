3、为了判断keeper上方的B方向是否形成off-state的小循环，给H4-H11的每个霍尔以自身为原点规定一个180°的工作面，若B方向处在工作面内，则认为该hall符合小循环的规律，为了壁面扰动，当5个或以上的hall满足这个条件时，输出第三个标记1（满足），否则输出0（不满足），其中，6号，5号hall工作面为以自身为原点的2，3象限，8号7号hall则为12象限，10号9号为14象限，11号和4号为34象限。
